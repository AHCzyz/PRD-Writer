- **状态管理层**：以 `src/store/editor-store.ts` (Zustand) 为核心，管理 `TabMLDocument` 结构、焦点状态及视图模式，提供行/列增删改查及序列化同步接口。
- **核心领域层**：`src/core/tabml/` 实现自定义 Tab-ML 格式的 AST 定义 (`types/tabml.ts`)、文本到 AST 的解析 (`parser.ts`) 及 AST 到文本的序列化 (`serializer.ts`)。
- **UI 表现层**：采用 `AppShell` 布局，通过 `viewMode` 在 `Grid` (HTML Table + 虚拟滚动) 和 `SourceView` (Monaco Editor) 间切换；`Grid` 组件集成 `useKeyboardNavigation` 实现 Excel 风格的键盘交互。
- **基础设施层**：`src/core/io/file-handler.ts` 封装 File System Access API，实现 `.tab.md` 文件的原生读写与降级兼容。
